1. This project has two end points for "product-detail" and "product-list" services.
2. Clone the projects form git hub.
3. Import the projects into STS and start the service
4. Hit the below urls to get the response for both the services
http://localhost:8021/api/product-detail/v1?skuId=APPLE_CAM
http://localhost:8021/api/product-list/v1
